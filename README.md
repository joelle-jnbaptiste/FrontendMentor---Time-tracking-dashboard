## ğŸ”® About The Project

This project is a solution to the **Time Tracking Dashboard** challenge from **Frontend Mentor**.

The goal was to build a **responsive dashboard** that allows users to:
- View time tracking statistics
- Switch between **Daily**, **Weekly**, and **Monthly** views
- Experience smooth hover states and clear visual hierarchy

This project focuses on:
- Component-based architecture with Angular
- UI composition using Ionic components
- State-driven UI updates
- Responsive layout using modern CSS techniques

## ğŸª„ Getting Started

This project was built using **Angular CLI** with **Ionic**, and deployed on **GitHub Pages**.

### ğŸ“˜ Prerequisites

- Node.js
- npm
- Angular CLI
- Ionic CLI

### ğŸ§™ Installation

1. Clone the repository:

       git clone https://github.com/joelle-jnbaptiste/FrontendMentor---Time-tracking-dashboard.git

2. Install dependencies:

       npm install

3. Run the development server:

       ionic serve

4. Open your browser at `http://localhost:8100`
