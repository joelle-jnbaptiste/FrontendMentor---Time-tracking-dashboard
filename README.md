## üéØ About The Project

This project is a solution to the **Time Tracking Dashboard** challenge from **Frontend Mentor**.

The goal was to build a **responsive dashboard interface** that allows users to:

- View time tracking statistics
- Switch between Daily, Weekly, and Monthly views
- Experience clear visual hierarchy and smooth hover states

This project focuses on:

- Component-based architecture with Angular
- UI composition using Ionic components
- State-driven UI updates
- Responsive layouts using modern CSS techniques

It demonstrates the ability to build a **data-driven dashboard UI** from a static design specification.

## ‚öîÔ∏è Getting Started

This project was built using **Angular CLI** and **Ionic** and can be run locally.

### Prerequisites

- Node.js
- npm
- Angular CLI
- Ionic CLI

### Installation

1. Clone the repository:

       git clone https://github.com/joelle-jnbaptiste/FrontendMentor---Time-tracking-dashboard.git

2. Install dependencies:

       npm install

3. Run the development server:

       ionic serve

4. Open your browser at:

       http://localhost:8100
